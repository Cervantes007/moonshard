#### Guide for Developers

1. Get the repo and install dependencies
```
$ git clone https://github.com/cervantes/moonshard.git
$ cd moonshard
$ yarn install
```

3. Available scripts
```
$ yarn dev
$ yarn build
$ yarn lint
$ yarn test
$ yarn test:coverage
$ yarn docs:dev
```
